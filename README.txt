1. main_settings.h - основные глобальные переменные, необходимые программе
2. SinglePassApp.h - основной модуль, окно, в котором происходит одобрение заявки, отклонение заявки,
отметка об использовании заявки, отмена заявки и т.д.
3. OrderPassApp.h - модуль заказа и редактирования пропусков
4. MessengerApp.h - окно обмена сообщениями, рудимент, который не был удалён до перехода к написанию WEB-программы
5. Application.glade - информация об интерфейсе. Составлена автоматически с помощью редактора форм GLADE.

Как должна была бы использоваться программа: на сервер помещается база данных, в программе указываются адрес сервера
и порт базы данных, программа компилируется и затем инсталируется на компьютеры пользователей.
